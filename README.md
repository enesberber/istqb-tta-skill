# ISTQB Technical Test Analyst (CTAL-TTA v4.0) Skill

A skill for AI coding agents (Claude Code, Cursor, Copilot, etc.) that provides ISTQB Certified Tester Advanced Level – Technical Test Analyst knowledge: white-box techniques, static/dynamic analysis, non-functional testing, risk-based testing, reviews, and test automation.

## Install

```bash
npx skills add https://github.com/enesberber/istqb-tta-skill --skill istqb-technical-test-analyst
```


## Contents

- **SKILL.md** — Main skill entry point and decision trees
- **rules/** — 24 rule files (one per syllabus concept)
- **command/** — 3 workflows: design white-box tests, plan non-functional tests, conduct technical review
- **references/** — Syllabus mapping and glossary

Based on the [ISTQB CTAL-TTA v4.0](https://www.istqb.org/) syllabus.
